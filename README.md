# 🏭 Content Factory Landing

Лендинг продукта **«Контент-завод под ключ»** — персональной ИИ-системы производства контента для экспертов и блогеров.

## 📦 Назначение

- **Продающая страница** основного оффера: AI-система автоматизации контент-производства
- **Целевая аудитория**: эксперты и авторы контента (30–50 лет, 70% женщины)
- **Ценностное предложение**: полный цикл создания и публикации контента без ручной рутины

## 🎯 Что внутри

- Демонстрация возможностей системы (парсинг конкурентов, генерация идей, оформление)
- Форма сбора контактов и запросов на демо
- Интеграция с основным воронком продаж

## 🛠️ Стек

- **Frontend**: React + Vite
- **Стилизация**: Tailwind CSS
- **Хостинг**: Vercel
- **CMS/Backend**: интеграция через API

## 🚀 Быстрый старт

```bash
# Клонирование
git clone https://github.com/ungurenko/content-factory-landing.git
cd content-factory-landing

# Установка зависимостей
npm install

# Разработка
npm run dev

# Продакшн
npm run build
npm run preview
```

## 📝 Структура проекта

```
content-factory-landing/
├── src/
│   ├── components/       # React-компоненты лендинга
│   ├── pages/            # Основные страницы
│   ├── styles/           # Tailwind + кастомные стили
│   └── App.tsx          # Главный компонент
├── public/              # Статика (изображения, фавиконка)
├── index.html          # Точка входа
└── vite.config.ts      # Конфиг Vite
```

## 🎨 Основные секции

- **Hero** — заголовок, CTА (призыв к действию)
- **Features** — преимущества системы
- **Demo** — демонстрация результатов
- **Testimonials** — отзывы пользователей (если доступны)
- **CTA Form** — форма для контактов и записи на демо
- **Footer** — навигация и соцсети
