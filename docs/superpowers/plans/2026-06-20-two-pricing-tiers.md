# Два тарифа на лендинге — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Добавить на лендинг второй, доступный тариф «Сам» за 9 990 ₽ рядом с «Под ключ» 24 990 ₽, выделив «Сам» как главный, и перевесить ценовые якоря страницы на «от 9 990 ₽».

**Architecture:** Все правки в одном файле `index.html` (HTML и CSS внутри одного документа). Блок цены превращается из одной карточки в две в общей сетке. Меняется ещё пять мест, где раньше фигурировала единственная цена 24 990 ₽ или старые формулировки.

**Tech Stack:** Статичный HTML + inline CSS, без сборки и фреймворков. Проверка — поиск по файлу (grep) и визуальный осмотр в браузере.

## Global Constraints

- **Ноль длинных тире (—)** в любом новом видимом тексте. Замены по таблице в `CLAUDE.md` проекта (запятая, двоеточие, глагол). Существующие 57 тире — в комментариях кода, их не трогаем.
- **Короткое тире (–)** только в числовых диапазонах (`2–5 минут`). Минус/дефис в «-50%» — обычный дефис, как в текущем коде.
- **Никаких гарантий результата** в тексте.
- **Чек/договор и юр.статус не упоминаем.** Подвал не трогаем.
- **Коммиты не делаем по ходу работы.** В конце, по команде Александра, одним или несколькими коммитами.
- Ссылка обеих кнопок-CTA — существующая `https://t.me/m/N0bTmb18OTUy` (тариф клиент называет в переписке).

  - CSS блока цены: `.pricing-grid` (стр. 2031–2037), штамп `.pricing-card--featured::after` (стр. 2086–2087).
