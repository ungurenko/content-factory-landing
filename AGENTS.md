# AGENTS.md — Контент-завод лендинг

## ⚠️ README неактуален

README описывает стек React + Vite + Tailwind, которого **нет**. Реальность: один файл `index.html` с ванильным HTML/CSS/JS.

## Структура: один файл, три секции

`index.html` (~2900 строк) всегда в порядке:

1. **`<style>`** (строки 12–1962) — весь CSS, desktop-first
2. **`<body>`** (строки 1964+) — HTML-контент
3. **`<script>`** (строки 2833+) — IntersectionObserver scroll-reveal + FAQ-переключение

При правках сохраняй этот порядок. Не выноси стили и скрипты в отдельные файлы без явного запроса.

## Именование блоков

Пронумерованные комментарии разделяют 16 блоков в CSS и HTML:

```html
<!-- ═════ BLOCK 1: Hero ═════ -->
<!-- ═════ BLOCK 2: Problem ═════ -->
...
<!-- ═════ BLOCK 16: Footer ═════ -->
```

При добавлении нового блока используй такой же комментарий с номером и названием.

## CSS-конвенции

- **BEM-like классы**: `.hero__left`, `.hero__title`, `.features-grid`
- **Кастомные свойства**: все в `:root` (цвета, отступы, радиусы, тени, transition)
- **Медиа-запросы**: в конце `<style>`, после всех блоков и `.reveal`-правил
- **Состояния**: `:focus-visible`, `:user-invalid`, `[aria-expanded]`, `[aria-invalid]`
